# 🤖 智能答题助手 - 图形化界面版

基于图像识别的自动答题脚本，提供友好的图形化操作界面。

## ✨ 功能特性

### 核心功能
- 📸 **屏幕截图** - 支持全屏或自定义区域截图
- 🔍 **OCR 识别** - 支持 PaddleOCR/Tesseract/EasyOCR 三种引擎
- 📝 **题目解析** - 自动识别单选题、多选题、判断题、填空题
- 💡 **答案匹配** - 精确匹配 + 关键词模糊匹配
- 🖱️ **自动点击** - 识别后自动点击正确答案选项
- 🔄 **连续答题** - 支持自动循环答题模式

### GUI 界面功能
- 🎨 **现代化界面** - 清晰直观的操作面板
- 📊 **实时显示** - 题目内容、选项、答案实时展示
- 📋 **运行日志** - 彩色日志记录所有操作
- ➕ **答案管理** - 可视化添加/删除答案
- 🖼️ **图片预览** - 截图后可预览查看
- ⚙️ **灵活配置** - OCR 引擎、截图区域、答题间隔可配置

## 📦 安装依赖

```bash
# 基础依赖
pip install Pillow tkinter

# OCR 引擎（至少安装一个）
pip install paddlepaddle paddleocr  # 推荐，中文效果好
# 或
pip install pytesseract tesseract-ocr
# 或
pip install easyocr

# 自动点击功能
pip install pyautogui
```

## 🚀 快速开始

### 启动图形界面
```bash
python auto_answer_bot_gui.py
```

### 使用流程

1. **初始化**
   - 选择 OCR 引擎（推荐 PaddleOCR）
   - 点击【🚀 初始化】按钮

2. **截图答题**
   - 确保答题界面可见
   - 可选：勾选"使用自定义区域"设置截图范围
   - 点击【📸 截取屏幕】预览
   - 点击【🎯 识别并答题】自动识别并点击答案

3. **自动模式**
   - 勾选【🔄 连续答题模式】
   - 设置答题间隔时间（秒）
   - 点击【▶️ 开始自动】启动循环答题
   - 点击【⏹️ 停止自动】停止

4. **答案管理**
   - 识别后点击【➕ 添加答案】手动录入
   - 通过菜单【设置】→【答案数据库管理】查看/删除

## 🎯 界面说明

### 控制面板
- **OCR 引擎**: 下拉选择识别引擎
- **初始化**: 启动答题机器人
- **截取屏幕**: 捕获当前屏幕
- **自定义区域**: 设置截图坐标和大小
- **自动点击**: 启用/禁用自动点击
- **连续模式**: 启用/禁用循环答题
- **手动答案**: 强制指定答案选项（A/B/C/D）

### 显示区域
- **📝 题目信息**: 显示识别的题目内容、类型、选项
- **💡 答案信息**: 显示匹配到的答案
- **📋 运行日志**: 彩色日志记录操作历史

### 操作按钮
- **🎯 识别并答题**: 执行一次完整答题流程
- **➕ 添加答案**: 手动添加答案到数据库
- **📂 从图片答题**: 从本地图片文件答题
- **▶️ 开始自动**: 启动/停止自动循环
- **🗑️ 清空**: 清空显示内容

## 💡 使用技巧

1. **首次使用**: 首次运行会下载 OCR 模型，请耐心等待
2. **截图区域**: 建议设置固定区域提高识别速度和准确度
3. **答案积累**: 多使用"添加答案"功能积累题库
4. **模糊匹配**: 系统会自动提取关键词进行模糊匹配
5. **手动干预**: 识别错误时可用"手动答案"强制指定

## ⚠️ 注意事项

- 确保答题界面清晰可见，无遮挡
- 屏幕分辨率不宜过高或过低
- 自动点击功能需要授予辅助功能权限
- 连续答题时注意间隔时间，避免过快
- 本工具仅供学习研究使用

## 🔧 常见问题

### Q: 初始化失败？
A: 检查是否安装了对应的 OCR 引擎库，首次使用需要下载模型文件。

### Q: 识别准确率低？
A: 尝试切换 OCR 引擎，PaddleOCR 对中文支持最好。

### Q: 自动点击无效？
A: 检查是否安装了 pyautogui，某些系统需要特殊权限。

### Q: 如何添加更多答案？
A: 使用界面中的"添加答案"按钮，或直接编辑 answer_database.txt 文件。
